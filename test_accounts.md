# 游牧岛 (YMD) 测试账号文档

本文档记录了项目中用于开发和测试的默认账号信息。这些账号可以通过运行 `ymd-server/seed_test_accounts.py` 脚本自动生成或重置。

## 管理员账号

用于登录后台管理系统 (ymd-admin) 或在客户端体验超级管理员权限。

| 角色 | 昵称 | 登录邮箱 | 密码 | 权限说明 |
| :--- | :--- | :--- | :--- | :--- |
| **超级管理员** | 测试管理员 | `admin@ymd-test.com` | `Admin1234!` | 拥有所有管理后台权限及前端所有特权 |

## 如何重置或生成测试账号

如果在开发过程中发现测试账号丢失、密码被修改或需要重新初始化，可以在 `ymd-server` 目录下执行种子脚本：

```bash
cd ymd-server
python seed_test_accounts.py
```

执行后，脚本会自动在数据库中创建或更新（如果已存在）这些测试账号，并输出执行结果。
